Git histogram

Generate a git log from your repository by using this command:

git log --since="2022-06-01" --numstat > extract.log

this will give you the commits since 01 June 2022 and for each commit you will have:
  • commit hash
  • merge (optional)
  • commit date
  • author
  • list of files affected and the change count
